# 📝 MERN Blog App

A simple blog application built using the **MERN stack** where users can read, like, and comment on tech blogs.

## 🚀 Features

- User authentication (register/login)  
- Create and read blog posts  
- Like and comment on posts  
- Tech-focused blogging platform  


## 🛠️ Tech Stack

- **MongoDB** – Database  
- **Express.js** – Backend  
- **React.js** – Frontend  
- **Node.js** – Server  



## ⚡ Run Locally

```bash
git clone https://github.com/Ankur-Dwivedi22/MERN-Blog.git
```

## 🎯 Purpose

To build a full-stack blogging platform and strengthen MERN stack development skills.
